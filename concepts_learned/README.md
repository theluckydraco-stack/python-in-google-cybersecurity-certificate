# Concepts Learned

Core Python concepts and fundamentals learned through the Google Cybersecurity Professional Certificate.

## Structure

- `practice.py` — Main learning script with all exercises
- `exercises.py` — Modular exercise functions
- `file_handling.py` — File I/O and pathlib concepts
- `data/` — Test data files for exercises

## Concepts Covered

1. **Failure Rate Calculations** — Functions and return statements
2. **Reading Rate Analysis** — Conditional logic and comparisons
3. **Login Behavior Analysis** — Data analysis and ratios
4. **String Operations** — Type casting, slicing, indexing, `.index()` method
5. **List Manipulation** — Membership validation, access control logic
6. **Regular Expressions** — Pattern matching, extraction, log scanning

## Running Exercises

```bash
# Run main practice script
python concepts_learned/practice.py

# Run specific exercises
python concepts_learned/exercises.py
```

## Learning Path

Start with `practice.py` for sequential learning through concepts 1-6.

Each section builds on previous concepts and includes practical examples.

## Data Files

Sample data files in `data/`:
- `allow_list.txt` — Access control example
- `login.txt` — Login log example

## Notes

Exercises progress from simple to more complex, introducing new concepts gradually.
